# 📊 نظام إدارة الفواتير الذكي | Smart Sales Analytics

نظام ويب متكامل يهدف إلى أتمتة إدخال الفواتير وتحليل المبيعات باستخدام تقنيات الذكاء الاصطناعي ومعالجة الصور، مما يوفر وقت وجهد المحاسبين ويقلل الأخطاء البشرية.

## 🌟 المميزات الرئيسية
* **التحويل الرقمي (OCR):** استخراج آلي لبيانات الفواتير (اسم المورد، المبلغ) من الصور.
* **لوحة قيادة تفاعلية (Dashboard):** عرض إجمالي الإيرادات، ومتوسط الطلبات، والموردين الأكثر تعاملاً.
* **إدارة السجلات:** بحث ذكي، فلترة سريعة (فواتير اليوم)، وإدارة متكاملة للسجلات.
* **تصدير التقارير:** إمكانية تصدير البيانات بصيغة `CSV` (Excel) أو طباعة تقارير `PDF`.
* **نظام الصلاحيات:** لوحة دخول مخصصة للمدراء (صلاحيات كاملة) ومدخلي البيانات (إضافة فقط).

## 🛠️ التقنيات المستخدمة
* **الواجهة الخلفية (Backend):** Python, Flask
* **الذكاء الاصطناعي ومعالجة الصور:** Tesseract OCR, OpenCV, TensorFlow
* **قاعدة البيانات:** SQLite
* **الواجهة الأمامية (Frontend):** HTML, CSS (تصميم متجاوب يدعم RTL)

## 🚀 طريقة التشغيل
1. قم باستنساخ المستودع (Clone).
2. ثبت المكتبات المطلوبة: `pip install -r requirements.txt`
3. تأكد من تثبيت محرك `Tesseract-OCR` على جهازك.
4. شغل النظام عبر الأمر: `python app.py`
